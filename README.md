# NestEase – Furniture & Appliance Rental Website

A landing page for a furniture and appliance rental service, built with **HTML, CSS, and JavaScript**.

## Features
- Hero section with call-to-action buttons
- Product catalog with category filters (Bedroom, Living Room, Kitchen, Appliances, Study)
- Rent vs. buy savings calculator
- Customer testimonials
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Files
- `index.html` – page structure
- `styles.css` – styling
- `script.js` – product rendering, filters, and calculator logic

## Usage
Clone the repo and open `index.html` in your browser.
